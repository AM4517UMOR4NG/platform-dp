Functions of HTML Tags

HTML Tag Description
<html> The main element that wraps all content in an HTML document.
<head> Contains meta information about the document, such as the title and links to stylesheets.
<body> The main content of the web page, where visible elements like text, images, and links are placed.
<div> A block-level container used to group other elements for layout or styling purposes.
<p> Defines a paragraph of text.
<a> Defines a hyperlink to another page or resource.
<img> Used to display images on a webpage.
<ul> Defines an unordered list (with bullet points).
<ol> Defines an ordered list (with numbers).
<li> Defines an item in an ordered or unordered list.
<strong> Defines important text, usually displayed in bold.
<em> Defines emphasized text, usually displayed in italics.
<h1> Defines the most important heading, usually displayed in large text.
<h2> Defines the second most important heading, smaller than <h1> but still large.
<span> Defines a section of text that can be styled or formatted inline.
